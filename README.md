# Lokale Entwicklung

Repository klonen:

```bash
git clone <repository-url>

Projektordner öffnen:

cd portfolio

Dependencies installieren:

npm install

Development Server starten:

npm run dev

Die Anwendung läuft unter:

http://localhost:3000
Docker Setup

Container starten:

docker compose up --build

Falls Port 3001 verwendet wird:

http://localhost:3001

Container stoppen:

docker compose down
